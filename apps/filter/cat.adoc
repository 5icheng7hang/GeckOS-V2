== NAME
cat - concatenate files and print on standard output

== SYNOPSIS
*cat* [OPTION]... [FILE]...

== DESCRIPTION
Concatenate FILE(s) to standard output.

With no file read standard input.

== EXAMPLES
*cat f g*::
	Output f's content, then g's content

*cat*::
	Copy standard input to standard output
