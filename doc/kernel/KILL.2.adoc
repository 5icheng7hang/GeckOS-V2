
= KILL(2)
:doctype: manpage

== NAME
KILL - terminate another task

== SYNOPSIS
Input::
	* AC - return code for task
	* XR - task ID to kill
Call::
	jsr KILL
Output::
	-

== DESCRIPTION
Ends a complete task, with all threads in them. 
AC is return code, XR is task ID to kill.
The return code is sent to the parent with SIGCHLD.
