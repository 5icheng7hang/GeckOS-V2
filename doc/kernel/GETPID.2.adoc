
= GETPID(2)
:doctype: manpage

== NAME
GETPID - get own task and thread ID

== SYNOPSIS
Input::
Call::
	jsr GETPID
Output::
	* XR - current task ID
	* YR - current thread ID

== DESCRIPTION
Returns the thread's own thread ID, and the task ID it belongs to.

== ERROR CODES

E_NOSTR::
	invalid stream number (not any of STDIN, STDOUT, STDERR, STDCERR)
